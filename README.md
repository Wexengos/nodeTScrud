# nodeTScrud
Simple CRUD done in Node.js (Express) with TypeScript and SQLite.

## running
Install yarn, 
Clone repository,
Run "yarn", then "yarn dev" on your terminal in the root folder.

In the root folder there is a JSON file for Insomnia testing.
